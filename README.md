# Wiki Feats

WikiFeats is a web app delivering Wikipedia's Featured Content.

## Technical Stack

- Common
  - Typescript
  - PNPM

- Front-End
  - React.js
  - Ant Design
  - React Query + Axios
  - Vitest + Testing Library

- Back-End
  - NestJS
  - MikroORM
  - PostgreSQL
  - Axios
  - NestJS Testing + Jest
