# Wiki Feats

WikiFeats is a web app that delivers Wikipedia's Featured Contents.

## Technical Stack

- Common
  - Typescript
  - NodeJS + PNPM

- Front-End
  - React.js
  - Ant Design
  - React Query + Axios
  - Vitest + Testing Library

- Back-End
  - NestJS
  - MikroORM
  - PostgreSQL
  - Axios
  - NestJS Testing + Jest
